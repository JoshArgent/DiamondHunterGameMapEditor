## Synopsis

Diamond Hunter game map viewer and editor.

## Installation

Run DiamondHunterMapViewer.jar to launch the map viewer.
Open .map files (one is provided as an example) and then launch the game by clicking File > Launch Game

## Dependencies

DiamondHunter.jar
jl1.0.1.jar
mp3spi1.9.5.jar
tritonus_share.jar

## Tests

Includes JUnit tests to test map loading functionality.
Look inside /DiamondHunterMapViewer/src/testing
